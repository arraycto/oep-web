### 在线考试项目-web端

&emsp;&emsp;毕设设计的Web端，技术选项来来回回花了很久，最后还是决定多锻炼一下自己。目前选型Vite+Vue3+Ts, 除Vue-router外尽量不用其它第三方组件,但会参考element的组件实现

------

部分目录结构介绍(针对于常规项目之外的)

pages:	网站页面布局及相应组件(侧标栏、头部等)

packages:	不针对于本项目的一些可复用的组件，如input、button、table、面包屑等
